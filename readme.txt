Auto Pin It Button adds a "Pin It" button to your blog posts automatically. 

== Description ==

Auto Pin It Button adds a "Pin It" button to your blog posts automatically, so the readers can pin it to Pinterest quickly. 

= Features =

*   Add a Pinterest Pin It Button to your blog posts.
*   Add a Pinterest Pin It Button over images on your site.
*	Pin your blog posts with backlink to your pinterest account automatically.(PRO)

== Installation ==

1. Login to your WordPress Dashboard.
2. Navigate to "Plugins" > "Add New".
3. Then click "Upload" in the header.
4. Choose the zip file you downloaded earlier.
5. Click "Install Now".
6. Once installed, tap on "Activate Plugin"
